# Shortly
## _Short link generator_
Bootstrapped with **expo**

- Insert a url
- Generate a short link
- View previously generated short links


## Tech
 - Expo SDK 45
 - NodeJS V14
 - React Native
 - Async Storage

## Installation

Install the dependencies and devDependencies, start the server and run in Expo.

```sh
cd shortly
yarn
yarn start
```
